# Real-Time Face Recognition and Identity Management System

## Overview

A real-time face recognition system built using Python, OpenCV, InsightFace and ONNX Runtime.

## Features

- Face Detection
- Face Recognition
- Confidence Scoring
- Unknown Face Detection
- Attendance Logging
- SQLite Integration

## Tech Stack

- Python
- OpenCV
- InsightFace
- ONNX Runtime
- SQLite
